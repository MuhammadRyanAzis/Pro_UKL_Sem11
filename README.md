# DevAcademy UKL

Platform E-Learning untuk proyek UKL DevAcademy.

## Struktur Proyek

- `backend/` - API NestJS dan database Prisma
- `frontend/` - Aplikasi frontend Next.js

## Instalasi

Di root:

```bash
npm install
npm run install:all
```

## Menjalankan

```bash
npm run dev
```

## Catatan

- `backend/` dan `frontend/` masing-masing memiliki `package.json` sendiri
- Tambahkan `frontend/README.md` jika perlu dokumentasi khusus frontend
- Pastikan `git remote add origin <url-repo-github>` jika belum diatur
